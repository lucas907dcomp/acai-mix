## User Story

**Como** usuário do AçaiMix,  
**Quero** um layout consistente com sidebar de navegação fixa e área de conteúdo,  
**Para que** a navegação entre PDV, Dashboard e Configurações seja fluida e padronizada.

---

## Acceptance Criteria

- [x] AC1: Componente `AppLayout` com sidebar fixa à esquerda e área de conteúdo principal
- [x] AC2: Sidebar exibe links de navegação baseados no role do usuário (Staff vê apenas PDV; Admin vê PDV + Dashboard + Configurações)
- [x] AC3: Sidebar exibe nome/role do usuário logado e botão de logout
- [x] AC4: Logout via `supabase.auth.signOut()` redireciona para `/login`
- [x] AC5: Layout responsivo: sidebar colapsa em mobile (hamburger menu + overlay drawer)
- [x] AC6: Rota ativa destacada visualmente na sidebar (NavLink isActive → bg-[#4c1e8c])
- [x] AC7: Cores seguem a paleta do design: sidebar bg-[#1a0b2e], borda [#2d1550], active [#4c1e8c]

---

## Scope

**IN:**
- Componente `AppLayout` com sidebar
- Navegação por role
- Logout funcional
- Layout responsivo básico (sidebar colapsável)

**OUT:**
- Conteúdo real das páginas (outras stories)
- Notificações ou badges de contagem
- Dark mode
