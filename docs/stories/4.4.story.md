## User Story

**Como** operador do caixa,  
**Quero** informar o valor recebido em dinheiro e ver o troco calculado automaticamente,  
**Para que** eu não erre o troco e o atendimento seja rápido.

---

## Acceptance Criteria

- [x] AC1: `CashFlow` em `src/components/pdv/CashFlow.tsx` aparece apenas quando `paymentMethod === 'cash'`
- [x] AC2: Botões de atalho com valores dinâmicos baseados no total
- [x] AC3: Campo de input numérico para valor recebido (teclado numérico em mobile via `inputMode="decimal"`)
- [x] AC4: Troco calculado em tempo real: `change = amountReceived - amount`
- [x] AC5: Troco exibido em destaque visual (fonte grande, cor verde)
- [x] AC6: Botão de confirmação fica desabilitado se `amountReceived < amount`
- [x] AC7: Valores dos botões de atalho gerados dinamicamente (próximos múltiplos acima do total)
