## User Story

**Como** operador do caixa,  
**Quero** ver as últimas vendas do turno na mesma tela do PDV,  
**Para que** eu possa verificar rapidamente se algo está errado sem sair da tela de caixa.

---

## Acceptance Criteria

- [x] AC1: `ShiftSalesTable` em `src/components/pdv/ShiftSalesTable.tsx` exibe as últimas 20 vendas do turno ativo
- [x] AC2: Colunas: horário (HH:mm), peso (g), valor (R$), método (PIX/Cartão/Dinheiro)
- [x] AC3: Nova venda aparece no topo da lista imediatamente após confirmação (invalidateQueries)
- [x] AC4: Ícone visual para `weight_source: 'manual'` (PenLine icon) para auditoria
- [x] AC5: Scroll interno quando lista ultrapassa a altura disponível
- [x] AC6: Exibe total parcial do turno no rodapé da tabela
