## User Story

**Como** operador do caixa,  
**Quero** ver claramente quando estou offline e quantas vendas estão pendentes de sync,  
**Para que** eu saiba que o sistema está funcionando e não perca dados.

## Dev Notes

**Estados visuais:**
```tsx
// Online + sem pendentes
<span className="text-xs text-green-600">● Online</span>

// Offline ou com pendentes
<Badge variant="destructive">
  ⚠ Offline — {pendingCount} venda{pendingCount !== 1 ? 's' : ''} pendente{pendingCount !== 1 ? 's' : ''}
</Badge>

// Sincronizando
<Badge variant="secondary">
  <Spinner size="xs" /> Sincronizando...
</Badge>
```

**isOnline false mas pendingCount > 0 enquanto online:** Pode ocorrer se drain falhou parcialmente. Exibir "X vendas pendentes" mesmo online para alertar.
