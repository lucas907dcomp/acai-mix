## User Story

**Como** operador do caixa,  
**Quero** que o total da venda seja calculado automaticamente ao capturar o peso,  
**Para que** eu não precise calcular nada manualmente.

---

## Acceptance Criteria

- [x] AC1: `saleStore` Zustand em `src/stores/saleStore.ts` com estado: `capturedWeightGrams`, `amount`, `paymentMethod`, `amountReceived`, `change`
- [x] AC2: Botão "Capturar Peso" lê `scaleStore.currentWeightGrams` e grava em `saleStore.capturedWeightGrams`
- [x] AC3: `amount` calculado automaticamente: `capturedWeightGrams × product.price_per_gram` arredondado para 2 casas
- [x] AC4: Preço por grama carregado do Supabase via React Query (tabela `products`, produto ativo da location atual)
- [x] AC5: `PriceDisplay` exibe: peso capturado, preço/g, e total em R$ com formatação brasileira
- [x] AC6: `reset()` limpa todos os campos após venda confirmada
- [x] AC7: Botão "Capturar Peso" desabilitado quando `currentWeightGrams` é null ou 0
