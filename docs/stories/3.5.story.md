## User Story

**Como** operador do caixa,  
**Quero** ver claramente o peso atual e o status da conexão da balança na tela do PDV,  
**Para que** eu saiba imediatamente se a balança está funcionando e qual o peso atual.

---

## Acceptance Criteria

- [ ] AC1: `ScaleConnectionStatus` em `src/components/pdv/ScaleConnectionStatus.tsx` — badge com 3 estados: "Conectada" (verde), "Desconectada" (vermelho), "Modo Manual" (amarelo)
- [ ] AC2: `WeightDisplay` em `src/components/pdv/WeightDisplay.tsx` — exibe peso em formato legível (ex: "345g" ou "1.250g")
- [ ] AC3: Peso atualiza em tempo real sem fazer a tela piscar ou re-renderizar outros componentes
- [ ] AC4: Quando provider é `ManualInputProvider`, `ManualWeightInput` (Story 3.3) aparece no lugar do `WeightDisplay`
- [ ] AC5: Fonte grande e legível para o peso — operador deve ler de ~50cm de distância

---

## Tasks

### T1 — ScaleConnectionStatus (0.5h)
- [ ] Badge Shadcn customizado com 3 estados visuais
- [ ] Lê `isConnected` e `providerType` do `scaleStore`
- [ ] Botão "Digitar Manualmente" integrado quando desconectada

### T2 — WeightDisplay (1h)
- [ ] Formata peso: `grams >= 1000 ? (grams/1000).toFixed(3) + 'kg' : grams + 'g'`
- [ ] Fonte grande: `text-5xl font-bold`
- [ ] Cor: verde quando peso > 0, cinza quando zero/null
- [ ] Animação suave na transição de peso (CSS transition)

### T3 — Conditional render (0.5h)
- [ ] Se `providerType === 'manual'`: mostrar `ManualWeightInput`
- [ ] Senão: mostrar `WeightDisplay`

## Dev Notes

**Zustand subscription seletiva:**
```typescript
// WeightDisplay só re-renderiza quando peso muda
const weight = useScaleStore(state => state.currentWeightGrams)
// NÃO usar: const store = useScaleStore() — causa re-render em QUALQUER mudança
```
