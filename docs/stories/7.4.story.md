## Dev Notes

**Recharts + Shadcn:** Recharts não é parte do Shadcn. Instalar separadamente. Para projetos futuros, considerar `shadcn/charts` (wrapper de Recharts), mas para simplicidade instalar direto.

**Dados para o gráfico:**
```typescript
// Por turno
const data = shifts.map(s => ({
  name: `Turno ${s.shift_number}`,
  value: s.total_amount,
  count: s.total_sales
}))

// Por dia
const data = dailySummary.map(d => ({
  name: format(d.sale_date, 'dd/MM', { locale: ptBR }),
  value: d.total_amount,
  count: d.total_sales
}))
```

**date-fns locale:** Instalar `date-fns/locale/pt-BR` para formatação de datas em português.
