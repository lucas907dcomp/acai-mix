## Tasks

### T1 — useDashboardMetrics (1h)
```typescript
// src/hooks/dashboard/useDashboardMetrics.ts
export function useDashboardMetrics(locationId: string, dateRange: DateRange) {
  return useQuery({
    queryKey: ['dashboard-metrics', locationId, dateRange],
    queryFn: () => fetchDashboardMetrics(locationId, dateRange),
    refetchInterval: 45_000,
    staleTime: 30_000,
  })
}
```

### T2 — useShiftHistory (0.5h)
```typescript
export function useShiftHistory(locationId: string, limit = 10) {
  return useQuery({
    queryKey: ['shift-history', locationId, limit],
    queryFn: () => supabase
      .from('shift_summary')
      .select('*')
      .eq('location_id', locationId)
      .order('opened_at', { ascending: false })
      .limit(limit),
    refetchInterval: 45_000,
  })
}
```

## Dev Notes

**DateRange type:**
```typescript
type DateRange = 'today' | 'week' | 'month' | { from: Date; to: Date }
```

**staleTime vs refetchInterval:** `staleTime: 30_000` evita refetch desnecessário quando o usuário troca de aba e volta em menos de 30s. `refetchInterval: 45_000` garante atualização periódica enquanto a aba está ativa.

**Queries do Supabase:** Usar as views criadas na Story 7.1. Não calcular aggregações no frontend.
