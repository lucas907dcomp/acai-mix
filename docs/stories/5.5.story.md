## Dev Notes

**Tempo decorrido:**
```typescript
import { differenceInHours, differenceInMinutes } from 'date-fns'

function elapsed(openedAt: string): string {
  const minutes = differenceInMinutes(new Date(), new Date(openedAt))
  const h = Math.floor(minutes / 60)
  const m = minutes % 60
  return h > 0 ? `${h}h ${m}min` : `${m}min`
}
```

**Totais:** Lidos de `shiftStore.activeShift.total_sales` e `total_amount` — atualizados por `shiftStore.updateTotals()` chamado em `saleStore.confirmSale()`.
