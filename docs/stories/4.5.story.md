## User Story

**Como** operador do caixa,  
**Quero** confirmar a venda com um clique e ter ela registrada imediatamente,  
**Para que** cada venda seja salva sem esforço e o histórico do turno seja atualizado.

- [x] AC2: Botão desabilitado até: peso capturado + método de pagamento selecionado + (se dinheiro) amountReceived >= amount

- [x] AC7: Loading state no botão durante o POST
