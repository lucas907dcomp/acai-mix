## Tasks

### T1 — Instalar libraries (0.5h)
- [x] `npm install zustand @tanstack/react-query react-router-dom dexie`
- [x] `npm install -D @tanstack/react-query-devtools`

### T2 — Configurar React Query no main.tsx (0.5h)
- [x] Criar `QueryClient` com `defaultOptions: { queries: { staleTime: 30_000, refetchOnWindowFocus: true } }`
- [x] Envolver `App` com `QueryClientProvider`
- [x] Adicionar `ReactQueryDevtools` em dev mode (`import.meta.env.DEV`)

### T3 — Configurar React Router (0.5h)
- [x] Criar `src/router/index.tsx` com `createBrowserRouter`
- [x] Rotas placeholder para `/login`, `/pos`, `/dashboard`, `/admin/settings` (lazy imports)
- [x] Envolver `App` com `RouterProvider` (substituiu `<App />` no main.tsx)

### T4 — Criar tipos globais (0.5h)

  - `ShiftStatus = 'open' | 'closed' | 'provisional'`
  - `WeightSource = 'scale' | 'manual'`
  - Interfaces: `Location`, `UserProfile`, `Product`, `Shift`, `Sale`
