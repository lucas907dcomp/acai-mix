## User Story

**Como** desenvolvedor do AçaiMix,  
**Quero** uma interface `IScaleProvider` com `MockScaleProvider` implementado,  
**Para que** o desenvolvimento do PDV possa avançar sem depender da balança física.

---

## Acceptance Criteria

- [x] AC1: Interface `IScaleProvider` definida em `src/providers/scale/IScaleProvider.ts` com métodos: `connect()`, `disconnect()`, `onWeight(callback)`, `onConnectionChange(callback)`, propriedade `type`
- [x] AC2: `MockScaleProvider` implementado em `src/providers/scale/MockScaleProvider.ts` com simulação de peso configurável
- [x] AC3: `MockScaleProvider` aceita parâmetro de peso fixo OU variação aleatória (para simular balança real)
- [x] AC4: `onWeight` retorna função `unsubscribe` para cleanup correto
- [x] AC5: `MockScaleProvider` pode ser criado com peso inicial e intervalo de emissão configuráveis
- [x] AC6: 7 testes unitários passando — emissão de peso, callback de conexão, unsubscribe, variância, peso não negativo
